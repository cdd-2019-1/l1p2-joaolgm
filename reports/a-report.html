Congressistas no Twitter

Com os dados da atividade dos parlamentares no twitter tenho interesse em saber se essa presena e atividade deles na rede notada, a partir da quantidade de curtidas e retweets (engajamento) a cada publicao, e tambm se existem parlamentares que sustentam essa fama atravs de retweets.

atividade = read_projectdata()

(5-) O quo comum os senadores de um estado terem maior engajamento do que os deputados?

validos = atividade %>%
  filter(!is.na(seguidores))

#glimpse(validos)

media_engaj_senador = validos %>%
  filter(casa == "senado") %>%
  group_by(UF) %>%
  summarise(media = mean(engaj_total))

#glimpse(media_engaj_senador)

engaj_deputados = validos %>%
  filter(casa == "cmara") %>%
  group_by(UF) %>%
  select(engaj_total)
## Adding missing grouping variables: `UF`
#glimpse(engaj_deputados)

ggplot() +
  geom_jitter(data = engaj_deputados,
              mapping = aes(x = reorder(UF, engaj_total), y = engaj_total),
              width = .3,
              alpha = .4) +
  geom_point(data = media_engaj_senador,
             mapping = aes(x = UF, y = media),
             color = "red",
             size = 2) +
  labs(x = "Estados",
       y = "Engajamento Total",
       title = "Deputados x Senadores")

Dos 27 estados em apenas trs - Mato Grosso do Sul, Pernambuco e Amap - existe uma certa diferena e que a mdia do engajamento dos senadores maior, em geral, do que a dos deputados.

(6-) Quantos parlamentares obtiveram seu maior engajamento com retweets?

engaj_rt = validos %>%
  filter(engaj_max == engaj_max_retweet)

#glimpse(engaj_rt)

engaj_rt %>%
  ggplot(aes(x = casa, y = engaj_max_retweet, color = casa)) +
  geom_quasirandom() +
  labs(title = "Engajamento com retweets",
       x = "Casa", 
       y = "Engajamento mximo com retweet", 
       color = "Casas")

Dos 608 parlamentares que obtemos nos dados, tiveram 163 que obtiveram seu maior engajamento a partir de retweet, sendo grande maioria destes com engajamento menor que 25000 reaes e a minoria deles acima, incluindo at, pontos isolados com mais de 75000 reaes em seus retweets.

(6.1-) Entre os que obtiveram maior engajamento, em quantos deles esse engajamento com retweet reflete como maior engajamento j obtido por ele?

top_engaj = engaj_rt %>%
  filter(engaj_max_retweet > 50000)

#glimpse(top_engaj)

ggplot() + 
  geom_point(data = top_engaj,
             mapping = aes(x = reorder(nome_eleitoral, engaj_max), y = engaj_max),
             color = "red",
             size = 3) +
  geom_point(data = top_engaj,
             mapping = aes(x = nome_eleitoral, y = engaj_max_proprio),
             color = "blue",
             size = 3) + 
  coord_flip() + 
  labs(y = "Engajamento mximo",
       x = "Parlamentar",
       title = "Tweet prprio x Retweet")

Com os pontos vermelhos simbolizando os retweets e os azuis os tweets prprios, entre os nove mais famosos do retweet - que alcanam mais de 50000 reaes cada um em seu retweet mais repercutido - nenhum deles conseguiu atingir tanto engajamento com seus tweets prprios e apenas uma parlamentar atingiu metade desse engajamento com tweet prprio.
